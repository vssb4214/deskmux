Set up DeskMux

Step 1 — Name this computer

Use a name you will recognize later, like “Gaming PC” or “Work laptop”.

Step 2 — Detect monitors

Find displays DeskMux can control on this machine. On Windows this uses native DDC; other platforms need shell commands in Advanced JSON.

Click Detect displays to list monitors, then capture each input value in the next step.

Step 3 — Capture input values

Switch the monitor to the input you want to capture, then click Read current input for that display. Label each capture so the generated config reads like your desk — for example “Desktop” or “MacBook”.

Step 4 — Generate config draft

Name your default preset, then build a starter config from the names and captured input values.

Step 5 — Validate and save

Open Advanced options below to review JSON, then validate and save from the desktop app.

Step 6 — Restart DeskMux

After saving, quit DeskMux completely and open it again. Your new configuration loads on startup.
